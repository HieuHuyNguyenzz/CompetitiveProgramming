# Một số trick xàm khi làm CP:
## 1. Sử dụng thư viện
```cpp
#inlcude<bits/stdc++.h>
```
Thư viện này có thể thay thế tất cả các thư viện STL khác, đỡ tốn thời gian~

## 2. Xuống dòng
Dùng
```cpp
cout<<"\n";
```
Thay vì
```cpp
cout<<endl;
```
